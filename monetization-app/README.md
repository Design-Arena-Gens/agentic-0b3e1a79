## Monetization Prompt Studio

Interactive workspace for crafting high-impact monetization prompts that can be dropped directly into AI copilots. Mix and match monetization levers, stage nuances, tone preferences, and success signals to generate an investor-grade brief tailored to your product.

### Features
- Guided form covering product context, revenue mix, and growth levers
- Strategy radar spotlighting high-leverage monetization opportunities
- Live prompt preview with copy-to-clipboard
- Responsive layout, optimized for Vercel deployment

### Tech
- Next.js App Router
- TypeScript + modern CSS modules

### Local Development
```bash
npm install
npm run dev
```

Visit `http://localhost:3000` to explore the prompt studio.

### Linting
```bash
npm run lint
```
